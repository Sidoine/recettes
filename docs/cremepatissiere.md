# Crème pâtissière (pour 125 ml)

## Ingrédients

- 125 ml de lait
- 12,5 g de maïzena
- 1 jaune d'oeuf
- Une gousse de vanille
- 15g de sucre fin

## Recette

- Chauffer le lait
- Dans le bol du mixeur,, mélanger tous les ingrédients
- Lancer le programme Expert 7 minutes, vitesse 7, 90°C
- Réserver au frais avant utilisation
